DDEESSCCRRIIPPTTIIOONN
       This  manual  page documents briefly the mmpp33cchheecckk command.
       This manual page was written for the Debian GNU/Linux dis-
       tribution  because  the  original  program does not have a
       manual page.

       mmpp33cchheecckk is a program that checks mp3  files  for  consis-
       tency  and  prints  several  errors and warnings. It lists
       stream attributes (color).  Layer 1,2,3,  mpeg1.0+2.0  are
       currently  supported.  CRC check for layer 3.  mmpp33cchheecckk is
       very useful for incomplete mp3 detection as it can be used
       to scan through your mp3 collection and find all mp3s that
       aren't perfect. Good for use with Napster and  other  bulk
       downloading of mp3s.

       These  programs  follow the usual GNU command line syntax,
       with long options starting with two dashes (`-').  Options
       can be specified in any order and mixed with files. Option
       scanning stops after a double dash  (--)  to  allow  files
       beginning  with  a dash.  A summary of options is included
       below.
